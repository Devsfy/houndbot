# Houndbot

A discord bot

## Getting Started

```bash
# install packages
npm install

# run
npm start

# run (with auto-restart)
npm watch
```

## Built With

* [DiscordJS](https://discord.js.org/#/)
* [Typescript](https://www.typescriptlang.org/)
